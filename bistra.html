Bistra Auspicious Occasions

Overview: A web-based platform to help users plan and organize special events like weddings, birthdays, and cultural ceremonies, making event management and vendor coordination simple.

Features

  • User registration and login
  • Event listings with details
  • Vendor booking and management
  • Calendar and reminders
  • Admin panel for managing events and vendors

Technologies Used

  • Frontend: HTML, CSS, JavaScript, DHTML, XHTML, Ajax
  • Backend: PHP, XAMPP, DHTML
  • Database: MySQL Server
  • Web Server: Apache Server
